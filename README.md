# Django Blog Project

A simple blog web application built with **Django**. Users can create, read, update, and delete blog posts. This project demonstrates basic **CRUD operations**, templates, URL routing, and Django admin usage.

---

## Features

- View all blog posts.
- Add, edit, and delete posts via Django admin.
- Simple and clean template design.

---

## Technologies

- Python 3.x
- Django 5.x
- SQLite (default database)
- HTML/CSS for templates
